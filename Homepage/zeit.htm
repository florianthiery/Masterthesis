Allen-Relationen

Relativ chronologische Beziehungen können über die von Allen beschriebenen Relationen zweier Events beschrieben werden.
Dieser Ansatz kann durch eine Erweiterung von Freksa (konzeptionelle Nachbarschaft) zur chronologischen Modellierung von
Beziehungen in der Samian Ware genutzt werden.



Der Linked Time Explorer

Der Linked Time Explorer zeigt exemplarisch eine Umsetzung der Allen/Freksa Relationen in eine RDF Ontologie und eine Visualisierung
als RDF-Graph, Zeitleiste und Text. Durch eine Modifikation können alle Relationen in Echtzeit auch in den Darstellungen geändert werden.
